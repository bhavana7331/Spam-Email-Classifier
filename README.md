# Spam Email Classifier

## Project Overview

This project classifies messages as Spam or Ham (Not Spam) using Machine Learning techniques.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* TF-IDF Vectorization
* Multinomial Naive Bayes
* Google Colab

## Dataset

SMS Spam Collection Dataset containing 5,572 messages.

## Methodology

1. Data Loading
2. Data Preprocessing
3. TF-IDF Feature Extraction
4. Train-Test Split
5. Naive Bayes Model Training
6. Performance Evaluation

## Results

* Model: Multinomial Naive Bayes
* Accuracy: 96.68%

## Example

Input:
"Congratulations! You have won a free iPhone."

Output:
Spam
